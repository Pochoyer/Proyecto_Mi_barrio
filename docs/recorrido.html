Pasos para iniciar el recorrido:

1. Selecciona una velocidad y pulsa Iniciar para comenzar el recorrido.
2. Puedes rotar la cámara con clic izquierdo, mover con derecho y zoom con la rueda.
3. Marca “Seguir cámara” para que el dron sea rastreado automáticamente.
